Jaatha

Jaatha is a composite-likelihood estimation method for models of evolution.

  • Available as CRAN package jaatha.
  • Uses genetic data from indiviuals of two distinct populations.
  • Uses simulations to approximate the likelihood function.
  • For each release a suite of benchmarks is performed.

Benchmark data

The raw benchmark results are available on Github. They consists of two datasets, one for measuring estimation accuracy and one for the runtime:

The Benchmark Explorer

The Benchmark Explorer is a Shiny App that allows to browse the benchmark data.

  • It's build using shiny, dplyr & ggplot2,
  • uses ggplot2's barcharts and boxplots to display the data,
  • and allows to interactively select whether to plot runtime or accuracy and displayed Jaatha versions and test models.
